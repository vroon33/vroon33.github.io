A Simplified Bird-Chirp Identifier!

This project was done as part of the M24 Signal Processing Project. The bird-chirp from a set of task files is mapped to a set of given reference bird-chirps, using the concepts of spectrograms and cross-correlations in MATLAB! View on GitHub

4-bit Carry Lookahead Adder

A CMOS-styled 4-bit carry look-ahead adder was created using NGSpice netlists, MAGIC for layout, and iVerilog, with the design optimized to meet specific delay requirements. Some other topologies are also used to reduce the delay. View on GitHub

Adaptive Filters and Vehicle Tracking
